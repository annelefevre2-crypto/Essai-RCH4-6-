RCH4-AI-OCR-FicheProduit
==============================
Application web (GitHub Pages) pour :
- Détecter automatiquement un cadre d'écriture délimité par 4 carrés noirs
- OCR uniquement de cette zone blanche
- Lecture du QR code présent dans le bandeau bleu
- Compilation OCR + QR dans un prompt RCH4 prêt pour ChatGPT

Déploiement :
1) Crée un dépôt public.
2) Upload tous les fichiers de ce ZIP à la racine.
3) Settings → Pages → Source : main / (root) → Save.
4) Ouvre l'URL en HTTPS (caméra requise).
